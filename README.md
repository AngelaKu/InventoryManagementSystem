# Inventory Management System

A Java-based console application for managing store inventory.  
Built using Object-Oriented Programming, file handling, and clean architecture.

---

## Features
- Add, remove, update products
- Search by ID
- Save inventory to CSV
- Menu-driven console interface

## How to Run
1. Open project in Eclipse
2. Run `InventoryApp.java`

---

## Security Notes

- This personal project is for demonstration and testing, not production use. So please do not use this exact code in anything seriously important.
